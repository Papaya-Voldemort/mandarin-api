Auth

POST /api/register, /api/login, /api/logout

Keys

Translate

POST /api/translate with x-api-key and JSON body

curl -X POST "$BASE/api/translate" \
  -H "x-api-key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text":"How are you?"}'
